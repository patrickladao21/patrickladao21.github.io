Loan Performance Insights Dashboard


Project Brief:

The Loan Performance Insights Dashboard project leverages Tableau and other dashboard tools to deliver in-depth insights into loan applications and funding metrics. It tracks key indicators like Total Loan Applications, Total Funded Amount, Total Amount Received, Average Interest Rate, and Average Debt-to-Income (DTI) Ratio across three sections. The summary section provides an overview of loan issuance and statuses, including Good Loans Issued, Bad Loans Issued, and Loan Status. The overview section analyzes loan applications based on various factors such as Month, State, Term, Employment Length, Purpose, and Home Ownership. The details section provides granular data on individual loan applications, enabling stakeholders to make informed decisions and improve lending processes.
